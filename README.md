# Tana Tech Africa

**Tana Tech Africa** is a digital solutions agency specializing in:

- Web & Mobile App Development  
- Social Media Management  
- Print Branding  
- Photography & Video Content Creation

This website is built using [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [TypeScript](https://www.typescriptlang.org), with a fully integrated blog powered by [Sanity CMS](https://www.sanity.io).
